# AP Crawler #

AP Exchange crawler & fetcher.

## Usage ##

    pip install -r requirements.txt
    cp config.yaml-dist config.yaml
    python fetch.py -h

## Requirements ##

- BeautifulSoup
- requests
- PyYAML
