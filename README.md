# AP Crawler #

Crawls and fetches content from the AP Exchange website.

## Usage ##

    pip install -r requirements.txt
    cp config.yaml-dist config.yaml
    python fetch.py -h

## Requirements ##

- BeautifulSoup 3.2.1
- requests 0.13.8
- PyYAML 3.10
